            <header>HTML Basics</header>
            <p>HTML stands for HyperText Markup Language.</p>
            <p>It is the standard language for creating web pages.</p>
            <p>HTML elements are represented by tags such as &lt;div&gt;, &lt;p&gt;, and &lt;a&gt;.</p>
            <p>These elements define the structure of a webpage.</p>
            <p>Attributes provide additional information about elements.</p>
            <p>For example, the href attribute is used in &lt;a&gt; tags.</p>
            <p>HTML documents must start with a <!DOCTYPE html> declaration.</p>
            <p>The &lt;html&gt; tag encloses all other elements on the page.</p>
            <p>Below is a simple example:</p>
            <code>&lt;!DOCTYPE html&gt;<br>&lt;html&gt;<br>&lt;head&gt;&lt;/head&gt;<br>&lt;body&gt;&lt;/body&gt;<br>&lt;/html&gt;</code>
            <p>Continue building your knowledge by writing your first HTML page!</p>
                      <h4> Fun Facts </h4>  
                        <ul>
                <li>HTML was created in 1991 by Tim Berners-Lee.</li>
                <li>It forms the backbone of all websites.</li>
                <li>HTML5 introduced new semantic elements.</li>
                <li>It's supported by all major browsers.</li>
                <li>HTML is case-insensitive.</li>
            </ul>

        <section class="main-section" id="CSS_Fundamentals">
            <header>CSS Fundamentals</header>
            <p>CSS stands for Cascading Style Sheets.</p>
            <p>It is used to style HTML elements on a webpage.</p>
            <p>CSS rules consist of selectors and declarations.</p>
            <p>Selectors target HTML elements to apply styles.</p>
            <p>Declarations include properties and values.</p>
            <p>Example:</p>
            <code>body {<br>    font-family: Comic Sans, sans-serif;<br>}</code>
            <p>CSS can be applied inline, internally, or externally.</p>
            <p>External stylesheets are linked using the &lt;link&gt; tag.</p>
            <p>CSS enables responsive design through media queries.</p>
            <p>For example:</p>
            <code>@media (max-width: 600px) {<br>    body {<br>        background-color: lightblue;<br>    }<br>}</code>
            <h4> Fun Facts </h4>
                        <ul>
                <li>CSS was introduced in 1996 by the W3C.</li>
                <li>It allows for separation of content and design.</li>
                <li>CSS3 added animations and transitions.</li>
                <li>CSS uses a box model for layout.</li>
                <li>It's compatible with all modern browsers.</li>
            </ul>
        </section>

            <p>Event listeners respond to user interactions.</p>

            <p>It is executed in the browser’s JavaScript engine.</p>
            <p>Learn more by writing and testing simple scripts.</p>
            <h4> Fun Facts </h4>
                        <ul>
                <li>JavaScript was created in 10 days in 1995.</li>
                <li>It runs on the client side for faster execution.</li>
                <li>JavaScript is single-threaded but asynchronous.</li>
                <li>Popular frameworks include React and Angular.</li>
